# 命名メモ: なぜ `lastshot` か

このディレクトリ `lastshot` の名前を決めた経緯を記録しておくドキュメントです。
（プロジェクトの内容・セットアップは `README.md` 側に書きます。ここは命名の由来専用。）

## このディレクトリの位置づけ

`eikodan` リポジトリの各サブプロジェクト（`rust-htmx` / `subsecond-demo` / `fastweb` /
`connectweb` / `pg-bench` / `playwright-sample`）で個別に試してきた要素を、
**全部統合した「最終進化形態」＝本番の実装**にあたるディレクトリ。

## 名前の物語（曳光弾 → 最後の一撃）

リポジトリ名 `eikodan`（曳光弾）は、本命を撃ち抜くために軌道を見せながら撃つ**試し撃ち**の弾。
各サブプロジェクトはその曳光弾＝照準合わせだった。

その照準の先で、**最後にこれで撃ち抜く本番の一発**がこのディレクトリ。
だから名前は `lastshot`（最後の一撃）。

```
eikodan（曳光弾＝試し撃ち・照準合わせ）  →  lastshot（最後の一撃＝本番・全部入り）
```

## 検討した方向と候補

「銃を撃つ系／絶対当てる・絶対撃ち抜く」のテーマで候補出しをした記録。

### 必中・百発百中系（＝絶対当てる）
- `hicchu`（必中）★和の本命候補 / `meichu`（命中） / `tekichu`（的中） / `hyakuhatsu`（百発百中）
- `sureshot` / `deadeye`（百発百中の射手・英名候補） / `deadshot` / `crackshot`

### 撃ち抜く・貫通系（＝絶対撃ち抜く）
- `kantsu`（貫通） / `tsuranuki`（貫き） / `penetrator` / `pierce` / `ap-round`（徹甲弾）

### 一撃・必殺系
- `ichigeki`（一撃） / `hissatsu`（必殺） / `oneshot` / `one-shot-one-kill` / `headshot`

### 射手・狙撃系
- `sniper` / `sharpshooter` / `marksman` / `crosshair` / `reticle`

### 照準・発射メカ系（開発トリガーとも掛かる）
- `lockon`（照準→撃ち抜く流れ） / `trigger` / `hammer` / `magnum` / `liveround`（実弾、曳光弾との対比）

## 最終候補の比較（`finalshot` → `lastshot`）

最後は「最終進化形態」の"最終"＋撃つ系の合わせ技で2つに絞った。

| 候補 | ニュアンス | 評価 |
| --- | --- | --- |
| `finalshot` | 「最終形・締めの本命」。やや造語っぽい | 意味は伝わるが語感が一段固い |
| `lastshot` | 「後がない、これで決める最後の一発」。英語の自然な言い回し | **採用** |

### `lastshot` を選んだ理由
- "last shot" は英語として完成された言い回し（ラストチャンス／一発逆転の最後の一発）で、
  `finalshot` より語感がカッコいい。短くて打ちやすい。
- 「試し撃ち（`eikodan`）を重ねて、最後にこれで撃ち抜く」という物語にぴったり合う。
- 既存サブプロジェクトの小文字一語の命名規則にも揃う。
